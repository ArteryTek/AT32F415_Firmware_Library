Release Notes for

AT32F415 Firmware Library Drivers

V2.0.6-2025/08/04

1.         GPIO

l  新增gpio_bits_toggle()函数。

2.         TMR

l  更新tmr_pwm_input_config()、tmr_encoder_mode_config()和tmr_output_channel_config()函数。通道模式配置前关闭通道使能，否则通道模式无法切换。

3.         USART

l  更新usart_flag_clear()函数，更改USART_RDBF_FLAG的清除方式为读DT寄存器清。

4.         更新部分注释。

 

V2.0.5-2024/12/13

1.         更新CAN驱动。

2.         更新USB驱动。

3.         更新部分注释。

 

V2.0.4-2024/08/08

1.         更新i2c_flag_clear和i2c_init函数。

2.         更新部分注释。

 

V2.0.3-2024/01/05

1.         完善部分驱动流程在IAR7.4下的编译警告。

2.         更新iar/gcc启动文件中ERTC对应的中断函数名。

 

V2.0.2-2023/10/26

1.         各IP驱动新增interrupt_flag_get函数。

2.         优化系统时钟切换流程。

 

V2.0.1-2023/08/04

1.         CRC驱动新增多项式及多项式宽度可修改的支持。

2.         更新了adc_ordinary_channel_set和adc_preempt_channel_set函数流程。

3.         更新部分注释。

 

V2.0.0-2022/11/18

1.         更新当HICK作为PLL时钟源时CRM_HICK48_NODIV作为默认设置。
